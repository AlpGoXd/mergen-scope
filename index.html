📊 Spectrum Analyzer .DAT File Viewer

Open and visualize .dat files from any spectrum analyzer — R&S ZNB, HP/Agilent, Keysight, and more. Free, browser-based, zero dependencies.

→ Open Viewer

Features

  • Multi-trace support – Load and compare multiple measurements
  • Interactive markers – Place and track peaks, identify frequency points
  • Noise PSD panel – Visualize noise density across spectrum
  • IP3/TOI measurement tool – 4-click intermodulation point calculation
  • Zoom & pan – Navigate large frequency ranges
  • Import/export – Save marked measurements and analysis

Supported File Formats

Compatible with spectrum analyzers and network analyzers from:

  • R&S ZNB series (ZNB8 and compatible)
  • Keysight and HP Agilent network analyzers
  • Additional formats via modular parser architecture

Technology

Vanilla JavaScript (ES6+) Recharts GitHub Pages GPL-3.0 License

Quick Start

  1. Click the "Open Viewer" button above
  2. Click Load File and select your .dat file
  3. Use markers to annotate points of interest
  4. Calculate IP3 or TOI in 4 clicks

Built for RF engineers and microwave researchers who need a quick, free tool to visualize spectrum measurements.
